# SPX Backtester
This strategy utilizes a series of iron condors, positioned closely but with slight variations, to help mitigate risk and reduce exposure. By specifying entry credit, stop limit, and stop loss, the entry and exit conditions of spreads can be fully automated.  

### How to Run Program
Download MatLab and open "UserInterfaceProto.mlapp". Insert option chain data in the "processed-data" directory.
